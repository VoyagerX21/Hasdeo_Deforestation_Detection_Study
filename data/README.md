# Hasdeo-Arand Forest Dataset (2018–2024)

This repository contains annual GeoTIFF datasets and model prediction outputs for the Hasdeo-Arand forest region, Chhattisgarh, India.

## Dataset Contents

### Annual GeoTIFF Files

* 2018.tif
* 2019.tif
* 2020.tif
* 2021.tif
* 2022.tif
* 2023.tif
* 2024.tif

### Prediction Outputs

The `hasdeo_outputs/` directory contains model prediction outputs and sample result patches generated from the GeoTIFF datasets.
